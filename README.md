## Installation #

Latest release version: `0.2.1`. See [RELEASE-NOTES.md](RELEASE-NOTES.md).

```xml
<dependency>

	<artifactId>spring-social-helper</artifactId>
	<version>0.2.1</version>
</dependency>
```
